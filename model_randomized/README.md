# FSP-model-dwarf-tomato
Functional–structural plant (FSP) model of dwarf tomato implemented in GroIMP to simulate 3D plant architecture, light interception, photosynthesis, and growth in vertical farming systems. Supports both static and dynamic simulations across a range of planting densities.

Note:
Some modifications has been applied by Alessandro De Vidi (GitHub: aledevv) for a digital twin project (traineeship) for University of Lincoln.
